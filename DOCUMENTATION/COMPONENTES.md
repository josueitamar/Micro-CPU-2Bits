Listado de Componentes

Listado detallado de los materiales necesarios para replicar el proyecto "Micro-CPU 2Bits".

| Cantidad | Componente | Especificaciones |
| :--- | :--- | :--- |
| 1 | Microcontrolador | **RP2040 Zero** (WaveShare o similar) con soporte MicroPython. |
| 1 | Protoboard | Estándar de 400 o 830 puntos. |
| 7 | Botones Pulsadores | Push-buttons de 2 o 4 pines (para dígitos y operaciones). |
| 6 | LEDs Monocromáticos | Preferiblemente: 4 de un color (Ciclo) y 2 de otro (Flags). |
| 1 | Display 7 Segmentos | **Cátodo Común** (1 dígito). |
| 7 | Resistencias | **220Ω a 330Ω** (para protección de segmentos del display). |
| 6 | Resistencias | **220Ω a 330Ω** (para protección de LEDs individuales). |
| 1 | Cable USB | Tipo USB-C para datos y alimentación. |
| Var | Cables Jumper | Macho-Macho y Macho-Hembra para conexiones. |
